Welcome to Charles

This wizard runs once and gets Charles set up on your machine. It should take about a minute.

Here's what will happen:

① Verify Python 3.10+ is installed
② Install Python packages for the API and voice service
③ Save your API keys to .env

🔒 Privacy note: Voice responses are synthesised by Microsoft Azure Neural TTS via the edge-tts library. The text of each response is sent to Microsoft's servers to generate audio. All other processing (wake word, speech-to-text, AI chat) runs locally or through your own API keys.

Connect Accounts

Charles needs two things: an AI provider and a wake word licence. Both are free.

Free at openrouter.ai/keys — sign in with Google or GitHub, then create a key
Free key at console.picovoice.ai — needed so the "Hey Charles" SDK can run
Free at virustotal.com — enables Charles to report whether a file or URL is malicious
