Getting Started
1
Run Charles.exe

Windows SmartScreen may appear — click "More info" → "Run anyway". Expected for unsigned installers.

2
Paste your API key

The setup wizard opens openrouter.ai automatically. Create a free account, copy your key, paste it in.

3
Click "Start Charles"

The launcher spins up the API service and voice pipeline. Ready in a few seconds.

4
Say "Hey Charles"

You're live. Try: "What security news dropped today?" or "Check for React CVEs this week."

Need an API key? Visit openrouter.ai/keys — no credit card required, takes about a minute to sign up. Once you're in the app you can pick any model available on OpenRouter — free or paid, fast or capable, whatever fits how you use Charles.
